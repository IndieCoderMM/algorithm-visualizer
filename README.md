# Algorithm Visualizer
This project is created for visualizing algorithms. *Visualizer Gui* is drawn using **Pygame Library**. 

This visualizer can illustrate the process of various **Sorting Algorithms**: 
1. Bubble Sort
2. Selection Sort
3. Insertion Sort
